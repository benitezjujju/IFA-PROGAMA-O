projeto desenvolvido na disciplina de progamação no itinerário de matemática.

A programação de computadores é a base invisível que sustenta a era digital moderna. Mais do que apenas escrever linhas de código, programar é a arte de traduzir problemas complexos em instruções lógicas que o computador possa executar, transformando ideias em softwares, aplicativos e automações funcionais. Este trabalho tem como objetivo explorar os conceitos fundamentais dessa ciência, destacando sua importância no desenvolvimento tecnológico atual e como a lógica de programação se tornou uma habilidade essencial para o futuro
